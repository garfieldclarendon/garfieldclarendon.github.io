Detailed Description

Handles a HTTP server socket request on a separate thread.

WebServerThread This class creates a QTcpSocket for a given socket descriptor. The run member creates the socket and parses the incoming HTTP header and attempts to find a UrlHandler for the url specified in the HTTP header.
