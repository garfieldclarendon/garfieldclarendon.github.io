Detailed Description

{put} /controller/module:serialNumber,address,classCode,action Message from the Block Module of state change in the monitored block. BlockStateChanged Block

{Number} serialNumber The controller's serial number (chip ID). {Number=0-8} address The sending module's address. {Number} classCode The sending module's classification. {Number} action The action requested. 0=Get,1=Add,2=Update,3=Delete Sent by a controller's BlockModule indicating the state of the block it's monitoring has changed. {json} Success-Response: { "action": "2" "messageUri": "/controller/module", "class": "8" "address": "0" "blocks": [ { "pin": 1 "pinState": 0 }, { "pin": 4 "pinState": 1 }, ] }
