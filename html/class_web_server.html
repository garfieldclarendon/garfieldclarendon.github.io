Detailed Description

An HTTP web server implementation that provides the REST API.

WebServer This class creates an HTTP web server based on a QTcpServer. Upon a tcp connection, the class looks for a url handler registered through createUrlHandler. If found, the handler's handleRequest member is called. A UrlHandler instance is created by calling WebServer::createUrlHandler for a given url. The web server calls the handleRequest member which emits the handleUrl signal.
